# Adidas Middle East Sales Performance Dashboard

An interactive Excel analytics dashboard visualizing sales performance, profitability, and consumer trends for Adidas across the Middle East region.

## 📊 Dashboard Overview
This project transforms raw transactional data into actionable business insights, tracking over **$85,069.85 in total profit**. It provides a comprehensive view of operational efficiency, regional demand, and purchasing behavior.

## 📈 Key Features & Visualizations
* **Geographic Analysis:** Bar chart tracking unit distribution across key markets (Iraq, Egypt, Oman, KSA, Lebanon).
* **Product Performance:** Pie chart breaking down profit by category (Footwear leads at 62%) alongside a Top 5 revenue-generating products chart.
* **Time-Series Trends:** Donut chart evaluating profit distribution by fiscal quarters.
* **Fulfillment Metrics:** Line graph mapping unit distribution across Online, Outlet, Retail, and Wholesale channels.
* **Financial & Demographic Insights:** Dual-axis chart correlating profit against transactional payment methods, paired with gender-based revenue distribution.
* **Interactive Slicers:** Dynamic filtering by Payment Method, Order Date, Product Category, and Store Type.

## 🛠️ Tech Stack
* **Software:** Microsoft Excel
* **Features Used:** Pivot Tables, Pivot Charts, Power Query (for data cleaning), Advanced Formulas, and Dashboard Slicers.

## 🚀 How to Use
1. Clone this repository.
2. Open the `.xlsx` file in Microsoft Excel.
3. Use the slicers on the top-right panel to filter data dynamically by year, category, or store type.
